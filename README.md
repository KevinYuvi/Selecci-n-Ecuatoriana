# Aplicación Móvil de la Selección Ecuatoriana de Fútbol

Este proyecto corresponde a una aplicación móvil básica desarrollada con **React Native** y **Expo Go**. La aplicación está basada en la Selección Ecuatoriana de Fútbol y presenta una pantalla de bienvenida con el logo de La Tri, además de una pantalla principal con información de los jugadores convocados.


## Tecnologías utilizadas

- React Native
- Expo Go
- JavaScript
- Visual Studio Code
- Node.js

## Componentes principales utilizados

La aplicación utiliza componentes básicos de React Native para construir la interfaz y permitir la interacción del usuario.

| Componente | Uso en el proyecto |
|---|---|
| View | Organiza los elementos visuales dentro de la pantalla. |
| Text | Muestra títulos, nombres, posiciones y datos de los jugadores. |
| Image | Presenta el logo de la Selección y las fotografías de los jugadores. |
| ScrollView | Permite desplazarse verticalmente por la pantalla principal. |
| FlatList | Muestra la lista de jugadores de forma ordenada. |
| TouchableOpacity | Permite seleccionar tarjetas y botones. |
| Modal | Presenta la información detallada de cada jugador. |

## Funcionalidades

- Pantalla de bienvenida con el logo de la Selección Ecuatoriana de Fútbol.
- Pantalla principal con la lista de jugadores convocados.
- Organización de jugadores por posición.
- Tarjetas individuales con fotografía, dorsal, nombre y posición.
- Ventana emergente con información detallada del jugador seleccionado.

## Instalación y ejecución

Para ejecutar el proyecto, se deben seguir los siguientes pasos:

1. Clonar el repositorio desde GitHub.

```bash
git clone https://github.com/KevinYuvi/Selecci-n-Ecuatoriana.git
```

2. Ingresar a la carpeta del proyecto.

```bash
cd primer_app
```

3. Instalar las dependencias necesarias.

```bash
npm install
```

4. Iniciar el proyecto con Expo.

```bash
npx expo start
```

5. Escanear el código QR desde la aplicación Expo Go en el celular o ejecutar el proyecto en el navegador web.
